# Proyecto Integrador - Análisis de Abandono de Clientes

## Descripción
Este proyecto analiza el abandono de clientes (churn) en una empresa de servicios, utilizando técnicas de Data Mining y Machine Learning.

## Objetivo
Identificar patrones en el comportamiento de los clientes para predecir posibles abandonos y mejorar la toma de decisiones.

## Archivos incluidos
- dataset_original.csv: Datos con valores faltantes y sin procesar
- dataset_limpio.csv: Datos tratados y preparados
- script_analisis.py: Script de análisis en Python

## Variables del dataset
- Antigüedad del cliente
- Tipo de plan contratado
- Uso de datos
- Número de llamadas
- Nivel de satisfacción
- Contactos con soporte
- Variable objetivo: abandono del cliente

## Proceso de limpieza
- Imputación de valores faltantes (media y mediana)
- Corrección de inconsistencias
- Preparación para análisis predictivo
